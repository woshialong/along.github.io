搭建一个这样的博客的笔记

发表于 2019-03-21 | 更新于 2019-03-22 | 分类于 学习笔记

这里记录了搭建这个博客的步骤，详见全文…

1.建立一个hexo网站

1.1环境准备

开发环境:Git+node+vscode的前端开发环境(请自行百度)

1.2项目创建

  • 打开cmd，执行 sudo npm install hexo-cli -g 命令，全局安装Hexo

  • 安装完后，执行 hexo -v 命令，验证是否安装完成

  • 在合适的文件夹，执行 hexo init projectname，初始化一个hexo项目

  • 进入文件夹，执行 cd projectname

  • 执行 npm install

至此已经成功创建了一个hexo项目

1.3项目基本操作

通过vscode打开你的hexo项目

  • 新建一篇文章

  • 生成静态页面

  • 本地启动

  • 打开链接可以看到如下页面

1.4项目配置设置主题

  • 在实际操作之前，建议去看它半个小时官方文档

  • 然后你需要配置一些个性化的config

  • 最后给你的博客选个主题并进行配置，这里我用的是next

2.部署到GitHub Pages

太长不想详细写了，直接写关键节点

  • 新建一个Github账号

  • 新建一个仓库，作为博客命名有要求：yourusername.github.io，同时需要public才能作为博客

  • 在setting里面开启GitHub Pages

  • 在config文件配置deploy配置项

  • 执行部署

  • 这时候你就可以通过 yourusername.github.io 打开了

3.完善域名和SSL证书

  • 在各大云服务平台选个域名买一下，要审核通过

  • 配置域名解析对应到 yourusername.github.io

  • 仓库setting里面配置Custom domain

  • 在各大云服务平台搞个免费的SSL证书，对应到你的域名上，也要审核通过

  • 仓库setting里面配置Enforce HTTPS

至此你就可以打开 ezrealong.cn 访问博客了
